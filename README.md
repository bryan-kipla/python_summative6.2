# Flask Productivity API (Session Auth)

## Project Description
This project is a secure Flask backend API built with session-based authentication.  
It allows users to register, log in, and manage their own personal notes.  
Each user can only access their own notes, and all routes are protected by authentication.  
The API supports full CRUD functionality and pagination on the notes index route.
